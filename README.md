# Microfrontend Host


# Getting started

1. Clone the repo
2. `yarn install`
3. `yarn start`

- the [`container`](https://github.com/MicrofrontendGenerator/container) Microfrontend Container
- the [`static`](https://github.com/MicrofrontendGenerator/static.git) Microfrontend Static

# Run the tests

- `yarn test`
